# 수도권대기환경청 AI 기반 대기오염물질 분석시스템

이동측정차량으로 수집한 대기 측정 데이터(VOC 및 PM2.5 등 복합 항목)를 지도에 시각화하고, 풍향·풍속 기반 간이 확산 모델로 **배출원 후보 사업장을 역추적**하는 웹앱입니다.

**서버·빌드·설치 불필요** — `index.html` 파일 하나를 브라우저로 열면 바로 동작합니다.

접속 화면에서 **접속 코드**를 입력해야 합니다 (기본값 `1111`, `index.html`의 `ACCESS_CODE` 상수에서 변경 가능). 새로고침 시 재접속이 필요하며 헤더의 로그아웃 버튼으로 언제든 잠글 수 있습니다. 클라이언트측 간이 잠금이므로 보안 목적이 아닌 접근 안내용입니다.

**PC·태블릿·모바일 반응형**을 지원합니다 (860px 이하에서 지도 상단 + 패널 하단 세로 배치로 자동 전환).

## 주요 기능

- **두 가지 측정 모드**
  - **VOC 측정**: 일반 VOC 측정기 데이터(성분별 ppb, TVOC 합산)
  - **PM·복합 측정 (Sniffer4D)**: Sniffer4DMapper CSV에서 PM2.5·PM10·PM1.0·NO2·VOCs·HCL을 자동 인식하고, 분석할 항목을 드롭다운으로 선택 (단위 µg/m³·ppm·mg/m³ 자동 표시)
- **사업장 확인 모드**: 측정 파일 없이도 선택 지역의 대기배출사업장만 지도에서 바로 확인
- 측정 데이터 지도 시각화 — 농도 백분위별 색상(빨강/주황/노랑/초록), 수천 지점도 빠른 canvas 렌더링
- 측정 지점 클릭 시 가우시안 플룸(또는 부채꼴)으로 바람 상류측 확산 범위 표시
- 확산 범위 내 대기배출사업장 자동 탐지 및 풍향 적합도 순위화 (후보 클릭 → 상세 정보 + 지도 이동)
- 농도 그래프 (측정 순서별 막대, 클릭 시 해당 지점으로 이동·확대)
- 항목별 평균/최대 통계
- 분석 보고서 내보내기 (인쇄/PDF) — 선택 항목·단위 반영
- 측정 원본 **TXT(탭 구분)·CSV·XLSX 모두 지원** — 인코딩(CP949/UTF-8)·헤더 위치 자동 감지
- 헤더의 **모드 변경 버튼**으로 시작 화면 복귀 → VOC ↔ PM·복합 ↔ 사업장 확인 모드 전환
- 지역 모드
  - **경기도**: 환경오염물질배출업소 공공 API 실시간 조회 (안산·시흥·평택·화성·김포·수원·오산, 대기배출시설 보유 업소)
  - **인천**: 대기배출사업장 1,824곳 내장 (산단 입주 1,111곳 건물 단위 정밀 좌표, 나머지 동 단위 근사) — 인터넷 없이 즉시 표출

## 사용법

1. `index.html`을 브라우저(Chrome/Edge 권장)로 열고 접속 코드를 입력합니다.
2. 시작 화면에서 **① 측정 데이터 종류**(VOC / PM·복합)와 **② 배출원 지역**(경기도/인천)을 선택합니다.
3. **③** 이동측정 데이터 파일(CSV·TXT·XLSX)을 드래그하거나 선택해서 엽니다. (PM·복합 모드는 업로드 후 분석 항목 선택)
4. 풍향·풍속을 입력하고(측정 당시 기상), 지도의 측정 지점 또는 농도 그래프 막대를 클릭합니다.
5. 사업장 데이터를 불러오면 확산 범위 내 배출원 후보가 순위와 함께 표시됩니다.

측정 파일 없이 사업장만 보려면 시작 화면의 **🏭 파일 없이 사업장만 보기** 버튼을 사용합니다.

### 측정 CSV 규격

**VOC 측정 모드**

- 헤더에 `No.`, `Latitude`, `Longtitude`, 성분별 `(ppb)` 컬럼, `합산값`(TVOC) 포함
- 상단 요약 블록(평균/최소/최대)이 있어도 자동으로 실제 헤더 행을 찾아 파싱

**PM·복합 측정 모드 (Sniffer4DMapper)**

- 3번째 줄이 헤더인 Sniffer4DMapper 원본 CSV
- `timeStamp, Longitude, Latitude, VOCs ppm, NO2 µg/m³, PM1.0/2.5/10 µg/m³, HCL mg/m³ ...` 컬럼을 자동 인식
- 측정 항목(PM2.5·PM10·PM1.0·NO2·VOCs·HCL) 중 분석 대상을 앱에서 선택
- 풍향·풍속은 우측 패널에서 직접 입력 (파일 내 wind 컬럼 미사용)

두 모드 모두 CP949(EUC-KR)·UTF-8 인코딩을 자동 감지합니다.

## 비용

**전부 무료입니다.**

| 항목 | 출처 | 비용 |
|---|---|---|
| 지도 타일 | CARTO light_all (OSM) | 무료 |
| 라이브러리 | cdnjs CDN (Leaflet, PapaParse, SheetJS) | 무료 |
| 경기도 배출업소 | 경기데이터드림 공공 API | 무료 |
| 인천 사업장 | 앱에 내장 (공공데이터 CSV 기반) | 무료 (호출 없음) |
| 인천 통계 | 공공데이터포털 API | 무료 |
| 지오코딩(선택) | Nominatim(무료) / Kakao REST(무료 쿼터) | 무료 |

참고: HTML에 포함된 경기도·공공데이터포털 API 키는 무료 발급 키로 과금이 없습니다. 다만 공개 저장소에 올리면 타인이 같은 키를 쓸 수 있으므로, 원하면 [경기데이터드림](https://data.gg.go.kr)에서 본인 키를 발급받아 `index.html`의 `GG_KEY` 상수만 교체하세요. Kakao 키는 코드에 포함되지 않으며 사용자가 직접 입력합니다(브라우저에만 저장).

## 주의사항

본 앱의 확산 표시(가우시안 플룸/부채꼴)는 **스크리닝 목적의 간이 모델**입니다. 풍향·대기안정도 입력값에 따라 결과가 달라지며, 규제·법적 판단에는 정식 확산모델(CALPUFF, HYSPLIT 등)과 현장 조사를 통한 검증이 필요합니다. 인천 사업장 좌표 중 일부(동 단위 근사)는 실제 위치와 차이가 있을 수 있습니다.
